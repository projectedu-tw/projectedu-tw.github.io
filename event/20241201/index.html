    20:00 +0230

  • 費用

    免費

立即了解

《114級 Project EDU 國立清華大學 清華學院學士班 模擬面試》正式啟動！

📍 活動形式: PEForum（https://forum.project-edu.org/）

👥 參加資格:

報考國立清華大學清華學院學士班並入選複試之特殊選才學生。

👩‍🏫 書審委員:

清華學院學士班及陽交大百川學程在學學長姐

📜 活動流程:

  1. 自我介紹 (1-3 min)
  2. 面試問答 (約12分鐘)
  3. 回饋與建議: 書審委員將針對每位學生的資料提供具體回饋與改善建議。（每位學員約15分鐘）

📝 報名方式:

  1. 加入PEForum
  2. 填寫報名問卷
  3. 於PEForum活動頻道，確認成功錄取審查場次及時間（預計於活動前24HR公布）
  4. 準時上線參加活動！
